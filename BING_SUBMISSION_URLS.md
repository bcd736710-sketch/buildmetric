## How to use this file

1. Open Bing Webmaster Tools.
2. Add or select the BuildMetric site.
3. Go to Sitemaps and submit the sitemap URL above.
4. Go to URL Inspection or Submit URLs and submit priority pages in the order below.
5. If Bing has a daily quota, submit Priority 1 first, then continue with the next groups on later days.
6. Do not submit the same URL every day unless Bing clearly says it was not submitted or was not discovered.
